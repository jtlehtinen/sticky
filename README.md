# sticky

Simple post-it notes for Windows.

## Keyboard shortcuts

    Toggle bold:          CTRL+B
    Toggle italic:        CTRL+I
    Toggle underline:     CTRL+U
    Toggle strikethrough: CTRL+D
    Decrease font size:   CTRL+[
    Increase font size:   CTRL+]
    Reset formatting:     CTRL+SPACE
